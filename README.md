## 테이블 구조


### 기능
* 회원 관리
* 혼자 라이딩
* **함께 라이딩 (방 CRUD, 참여, 방장 기능, 라이딩 시작 및 종료, 채팅)**
* 스탬프, 뱃지
* 관리자 페이지


### 함께라이딩에 사용된 기술
 
* 사용 언어 및 프레임워크
  * `Java`
  * `Spring Framework`
  * `node.js` 
  * `jsp`, `EL`
  
* 웹 표준 및 프론트
  * `HTML5`, `CSS3`, `Javascript`
  * `jQuery`
  * `Bootstrap4`

* DB
  * DBMS - `MySQL`
  * `MyBatis`
  
* API
  * `T Map Api` : 출발지-도착지 경로 검색, 현재 위치-목적지 거리 계산

* 배포
  * `AWS`의 `EC2`와 `RDS`
  * `FileZila`를 이용한 `FTP 업로드`
  * `Putty`를 이용한 `SSH 접속`
  * `CloudFlare`를 이용한 `보안접속(SSL)`
